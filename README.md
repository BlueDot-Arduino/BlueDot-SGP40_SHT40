# BlueDot-SGP40_SHT40
This library allows the user to program and simultaneously read the outputs of the SGP40 Sensor and the SHT40 Sensor with an Arduino Board.

* Source Files (.cpp and .h)
* Example Sketch: SGP40_SHT40_Test.ino
